  • 那究竟什么是“性能”呢？Web性能主要指网站的加载速度。

  • 加快内容的传输

缩短网站的加载时间是网站运营成功的重要因素。

当人们说“Web性能的重点是网站加载速度更快”时，他们的主要关注点是缩短加载时间。简言之，加载时间就是从用户请求网站到网站出现在用户屏幕上所经历的时间。
